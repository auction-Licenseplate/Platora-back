# 🚘 Module

####  - Auth Module : 사용자 인증 및 권한, 로그인 관리
####  - Users Module : 사용자 정보 관리
####  - Vehicles Module : 차량 번호판 정보 관리
####  - boards Module : 경매 생성, 입찰, 종료 관리
####  - Payment Module : 결제 처리 및 환불 관리
####  - OpenAI Module : 재미나이 API 관리
####  - Notification Module : 이메일 전송 및 알림 관리
####  - Admin Module : 관리자 기능 관리
